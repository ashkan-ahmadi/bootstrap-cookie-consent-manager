Cookie Conset Manager

See GitHub repo for more information and instructions on how to use.

If you do not see the cookie consent banner or modal on page load, you need to clear the local storage values and then refresh the page.

Remember that some adblockers such as uBlock Origin could cause interference with the GTM tags and not allow them to load properly.

You can see all the consent information on the Dev Tools > Application > Local storage (at the moment, everything is stored in the local storage. A future plan is to allow local storage or cookie).

It's very important to check the Console. There will be messages logged in the console informing you what consents are granted and denied. If you do not see any messages in the Console, disable your adblockers and/or use Incognito.

I recommend checking out the Google Tag Manager consent mode support. It has some useful information.
